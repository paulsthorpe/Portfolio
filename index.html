My Skills

I desire to be fully capable of handling any task involved in the process of creating web sites or web applications. In order to fulfill that desire I am constantly adding new tools to my arsenal. I use HTML, CSS, Javascript and Jquery to make things organized and sexy on the client facing side. Though not always, I do use bootstrap to help create layouts that will respond and adapt to different devices. By using bootstrap, a fluid layout is easily achievable and creates a reliable outcome.



If a database is required for the project I employ MYSQL to store all data, and use PHP to access data safely and efficiently. Object Oriented practices are used to make sure applications are efficient, reliable, easily manipulated and scalable. PHP is a reliable and time tested technology.
